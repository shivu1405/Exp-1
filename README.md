# Exp-1 ATM & BANKING SYSTEM

# AIM:
To develop an ATM & Banking System that automates basic banking operations such as cash withdrawal, deposit, balance inquiry, and fund transfer, ensuring secure and user-friendly transactions.
# SRS (Procedure):
1. Identify Requirements: Collect all functional and non-functional requirements of the ATM system.

2. System Analysis: Analyze user needs and define system functionalities like withdraw, deposit, and balance check.

3. Design Phase: Create system architecture, DFDs, and UML diagrams for process flow.

4. Database Design: Design tables for users, accounts, and transactions securely.

5. Coding: Develop individual modules for login, transactions, and account management.

6. Integration: Combine all modules and ensure they work together properly.

7. Testing: Test the system for accuracy, security, and performance.

8. Implementation & Maintenance: Deploy the system and perform regular updates and bug fixes.
